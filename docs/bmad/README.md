# BMAD-Methode

Dieses Projekt verwendet die **BMAD-Methode** (Business, Market, Architecture, Development) zur strukturierten Planung und Dokumentation. Die Methode umfasst Product Brief, PRD (Product Requirements Document), Architektur, Stories und Testplan.

## Dokumente (Reihenfolge)

| #  | Datei                    | Beschreibung                          |
|----|--------------------------|---------------------------------------|
| 01 | `01_product_brief.md`    | Product Brief – Vision & Zielsetzung  |
| 02 | `02_prd.md`              | Product Requirements Document (PRD)   |
| 03 | `03_architecture.md`     | Technische Architektur                |
| 04 | `04_stories.md`          | User Stories & Akzeptanzkriterien     |
| 05 | `05_test_plan.md`        | Testplan & Qualitätssicherung         |
